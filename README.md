# Electron Boilerplate

**Built with:**

- [CoffeeScript](http://coffeescript.org/)

- [Jade](http://jade-lang.com/)

- [SASS](http://sass-lang.com/)

- [BS4 (Alpha)](http://v4-alpha.getbootstrap.com/)

- [Backbone.js](http://backbonejs.org/)
